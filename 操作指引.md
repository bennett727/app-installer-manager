## 1. 软件简介

批量应用安装管理器是一款 Windows 平台的应用批量安装工具，主要功能：

- **批量安装** — 一键安装多个应用程序，自动检测静默参数
- **智能回退** — 静默安装失败自动切换正常安装
- **智能卸载** — 读取注册表信息，一键卸载已安装应用
- **开机自启管理** — 查看和删除系统开机自启项
- **完整日志** — 记录所有安装/卸载操作，可追溯

> 系统要求：Windows 7 及以上，无需安装 Python

## 2. 首次启动

### 2.1 放置程序

将 `AppInstaller.exe` 放在**非系统保护目录**，推荐路径：

```
D:\AppInstaller\AppInstaller.exe
```

> ⚠️ 不要放在 `C:\Program Files\` 或 `C:\Windows\` 等系统目录，否则可能因权限不足无法运行

### 2.2 启动程序

双击 `AppInstaller.exe` 启动，首次启动会自动创建 `data` 目录：

```
AppInstaller/
├── AppInstaller.exe
└── data/              ← 自动创建
    ├── packages/      ← 安装包存放
    ├── logs/          ← 日志文件
    └── config.json    ← 配置文件
```

![首次启动](images/02_first_launch.png)

### 2.3 放置安装包

将要安装的软件安装包（`.exe`、`.msi`、`.bat`）放入 `data\packages` 目录：

![放置安装包](images/03_place_packages.png)

## 3. 添加应用

### 方式一：扫描添加（推荐）

1. 将安装包放入 `data\packages` 目录
2. 点击菜单 **工具 → 扫描安装包** 或按快捷键 `Ctrl+S`
3. 程序自动识别安装包并添加到应用列表

![扫描安装包](images/04_scan_packages.png)
![扫描安装包](images/04-1_scan_packages.png)

### 方式二：手动添加

1. 点击 **"添加应用"** 按钮
2. 在弹出的对话框中填写信息：

| 字段 | 说明 |
|------|------|
| 安装包路径 | 点击"浏览"选择安装包文件 |
| 应用名称 | 自动识别，可手动修改 |
| 安装目录名 | 安装后的文件夹名称 |
| 安装路径 | 默认安装位置，可修改 |
| 安装参数 | 自动检测静默参数，可手动修改 |
| 复制安装包到本地 | 勾选后复制到程序目录（占用双倍空间） |

![添加应用对话框](images/05_add_app_dialog.png)

> 💡 **关于"复制安装包到本地"**：默认不勾选，程序直接引用原始路径。如果安装包在U盘或临时目录，建议勾选，避免安装时找不到文件。

### 修改应用参数

在应用列表中**双击**应用，或右键选择 **"编辑"**，可修改安装参数。

![编辑应用](images/06_edit_app.png)

## 4. 批量安装

### 4.1 安装单个应用

在"应用管理"标签页，选中应用后点击 **"安装"** 按钮。

### 4.2 批量安装

1. 切换到 **"批量安装"** 标签页
2. 勾选要安装的应用（默认全部勾选）
3. 点击 **"开始安装"**

![批量安装](images/07_batch_install.png)

### 4.3 安装过程

- 安装过程中会显示实时进度和日志
- **静默安装**：后台自动安装，无需操作
- **正常安装**：静默失败后自动切换，会弹出安装向导，按提示操作即可
- 安装完成后状态自动更新

![安装进度](images/08_install_progress.png)

### 4.4 重试安装

安装失败的应用，可在应用列表中选中后点击 **"重试"**。

## 5. 卸载应用

1. 切换到 **"已安装"** 标签页
2. 选中要卸载的应用
3. 点击 **"卸载"** 按钮
4. 确认卸载

![卸载应用](images/09_uninstall.png)

> 💡 程序会自动读取注册表中的卸载信息。如果显示"不可卸载"，说明该应用未在注册表中注册（如绿色软件），需手动删除。

---

## 6. 开机自启管理

1. 切换到 **"开机自启"** 标签页
2. 查看系统所有开机自启项
3. 选中要删除的自启项，点击 **"删除选中"**

![开机自启管理](images/10_startup_manager.png)

> ⚠️ 删除开机自启项后，对应程序将不再开机自动运行。部分更改需要重启电脑才能生效。

## 7. 操作日志

1. 切换到 **"日志"** 标签页
2. 可按应用名称筛选日志
3. 可设置显示条数
4. 点击 **"刷新"** 更新日志

![操作日志](images/11_logs.png)

日志记录了每次安装/卸载操作的结果和详细信息，可用于排查问题。

## 8. 设置与激活

### 8.1 设置默认安装路径

1. 切换到 **"设置"** 标签页
2. 在"默认安装路径"中输入或浏览选择路径
3. 点击 **"保存"**
4. 可选：点击 **"应用到所有应用"** 批量更新已有应用的安装路径

![设置页面](images/12_settings.png)

### 8.2 激活软件

1. 在设置页面点击 **"激活"** 按钮
2. 输入激活码
3. 点击确认

激活码类型：

| 类型 | 说明 |
|------|------|
| 月卡 | 有效期延长1个月 |
| 半年卡 | 有效期延长6个月 |
| 永久卡 | 永久有效 |

![激活对话框](images/13_activation.png)

### 8.3 查看有效期

在设置页面可查看当前有效期信息和系统信息。

## 9. 常见问题

### Q: 启动时提示"无写入权限"？

将程序移到非系统目录（如 `D:\AppInstaller\`），或右键选择 **"以管理员身份运行"**。

### Q: 添加应用时检测到的参数不正确？

添加后双击应用，手动修改安装参数。常见静默参数：

| 参数 | 适用安装包类型 |
|------|---------------|
| `/S` | NSIS 打包（如 7-Zip） |
| `/VERYSILENT /NORESTART` | Inno Setup 打包 |
| `/s /v/qn` | InstallShield 打包 |
| `/qn /norestart` | MSI 安装包 |

### Q: 安装失败怎么办？

1. 查看"日志"标签页中的错误信息
2. 常见原因：
   - 安装包损坏 → 重新下载
   - 参数不正确 → 手动修改参数
   - 权限不足 → 以管理员身份运行
   - 磁盘空间不足 → 清理磁盘
   - 安装包文件丢失 → 重新添加应用

### Q: 批量安装时有些软件弹出了安装向导？

这是正常的。程序先尝试静默安装，如果该软件不支持静默安装，会自动切换到正常安装模式，需要手动操作安装向导。

### Q: 卸载时提示"不可卸载"？

说明该软件是绿色版/便携版，没有在系统注册表中注册卸载信息。需要手动删除安装目录。

### Q: 如何在多台电脑上使用？

1. 将 `AppInstaller.exe` 复制到新电脑
2. 将安装包放入 `data\packages` 目录
3. 启动后重新扫描添加应用
4. 安装路径可能需要根据新电脑调整

### Q: 如何备份数据？

复制整个 `data` 目录即可。恢复时将 `data` 目录放回程序同级位置。
